# Лабораторная работа 5: Docker & PostgreSQL

Проект переведен на микросервисную архитектуру с использованием Docker и PostgreSQL.

## Структура
- **web**: Django приложение (порт 8000)
- **db**: PostgreSQL 15 (порт 5432, доступен только внутри сети Docker)

## Запуск проекта

### Требования
- Docker
- Docker Compose

### Инструкция для запуска (Dev/Prod)
1. Создайте файл `.env` на основе примера (см. ниже).
2. Запустите сборку и старт контейнеров:
   ```bash
